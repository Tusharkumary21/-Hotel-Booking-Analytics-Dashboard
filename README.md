# -Hotel-Booking-Analytics-Dashboard
Hotel Booking Analytics Dashboard

A Data Analytics project developed using SQL and Power BI to analyze hotel booking data and generate actionable business insights.

Project Objectives

- Analyze hotel booking trends
- Calculate cancellation rates
- Identify top booking countries
- Evaluate customer behavior patterns
- Compare hotel performance metrics

Tools Used

- SQL
- Power BI
- Excel/CSV Dataset

Dashboard Features

- KPI Cards (Total Bookings, Cancellation Rate, Average ADR, Lead Time)
- Booking Trend Analysis
- Cancellation Analysis
- Country-wise Booking Distribution
- Customer Type Segmentation
- Interactive Slicers and Filters

Key Insights

- Peak booking periods identified
- Higher cancellation rates in City Hotels
- Top contributing countries analyzed
- Transient customers dominate bookings
- Early bookings have greater cancellation probability
